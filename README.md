# Static FM — Late Night Weather Radio

Five weather modes. Curated playlists. Ambient audio. A DJ who's been on the midnight shift too long.

**[Tune in](https://static-fm.nowherelabs.dev)**

## Features

- 5 weather modes: rain, storm, fog, snow, clear night
- Curated playlists with Spotify integration
- Ambient weather audio (Web Audio API synthesis)
- Dynamic storm thunder synced to lightning flashes
- Fullscreen immersive mode
- Keyboard shortcuts (1-5 for weather, F for fullscreen)
- Listening history
- Shareable weather links (?weather=fog)

## Tech

Vanilla HTML/CSS/JS. Spotify IFrame API. Web Audio API for ambient sounds. Hosted on Vercel.
